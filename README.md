# 更新列表

## [1.2.0]

### 添加

- 固定装置：代表固定在世界上的可生成物的数据卡
- 外景固定装置：按钮、开关、控制杆、滑块、阀门
- 资产仓库拖放：将生成物和固定装置直接从仓库拖放到您的场景中！
- 电路：将固定装置连接在一起所需的线路
- 电路节点：添加、乘法、重新映射、值、翻转翻转、异或
- EntityPose：用于摆出实体姿势的数据卡，将其与电路固定装置一起使用以设置其初始状态
